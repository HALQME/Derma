# Installation
## Clone repo
```bash
git clone https://github.com/HALQME/Derma.git
```
or 
```bash
gh repo clone HALQME/Derma
```
### build
require swift (Operation has been confirmed only with 5.10.)
if you have not install swift, [see here](https://www.swift.org/install/)
```bash
# pwd path/to/Derma
make
```

# Uninstall
```bash
# pwd path/to/Derma
make uninstall
```
